get_closest_asteroids:
цель запроса:
список астероидов на основе их ближайшей даты приближения к Земле
параметры: ключ, дата начала периода, дата окончания периода
вывод запроса: ближайшие к Земле астероиды в заданный период

get_pic_of_the_day:
цель запроса:
вывести астрономическую картинку в определенный день
параметры: ключ, дата, параметр качества картинки (hd true/false)
вывод запроса: астрономическую картинку в определенный день

get_mars_weather_service:
цель запроса:
сводные данные по каждому Sol для каждого из последних семи доступных Sols (Sol - марсианский день)
параметры: ключ, формат выходного файла, версия API
вывод запроса: сводные данные по каждому Sol для каждого из последних семи доступных Sols (Sol - марсианский день)
